初回だけ、GitHub の Personal Access Token を登録するとボタン一発で公開できます。

トークンの取り方（クリックで展開）
  1. GitHub のトークン発行ページを開く
  2. Token name：Family Album など好きな名前
  3. Expiration：長め推奨（1 年など）
  4. Repository access：Only select repositories → kurigorira/home を選択
  5. Repository permissions → Contents を Read and write に設定
  6. Generate token → 表示された github_pat_... をコピー
  7. 下の欄に貼り付けて「保存」

または（手動アップ）

ZIP から手動アップする手順
1. ZIP を解凍
2. リポジトリ直下（photos/ や data/ がある階層）で「Add file → Upload files」
3. 解凍フォルダの中身（photos と data）をまとめてドラッグ&ドロップ
注意：GitHub 上ですでに photos/ に入った状態でドロップすると入れ子（photos/photos/）になります。必ずリポジトリ直下で行ってください。
